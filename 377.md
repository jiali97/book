### 377 Combination Sum IV

> 这道题使用DP来做。

```python
class Solution(object):
    
    def combinationSum4(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        dp = [0 for i in range(target + 1)]
        dp[0] = 1
        for i in range(len(dp)):
            for num in nums:
                if i >= num:
                    dp[i] += dp[i - num]
        return dp[target]
```
